CPU-Z는 컴퓨터의 **CPU, 메모리, 메인보드, 그래픽카드(GPU)** 등의 상세 하드웨어 정보를  
실시간으로 확인할 수 있는 무료 시스템 정보 프로그램입니다.  
CPU 클럭, 코어·스레드 구성, 캐시 구조, 메모리 종류 등 PC 내부 정보를 정확하게 분석해주기 때문에  
조립PC 검사, 게이밍 시스템 체크, 오버클러킹 모니터링에 널리 사용됩니다.

설치형 버전과 **무설치 Portable 버전**을 모두 제공하여 누구나 가볍게 실행할 수 있습니다.

## 📌 CPU-Z 주요 기능

- CPU 실시간 **클럭 / 배수 / 전압** 확인  
- 코어·스레드 구성, L1~L3 캐시 구조 표시  
- 메인보드 칩셋 / BIOS 버전 정보 제공  
- 메모리 클럭, 용량, 타입, 타이밍 정보 확인  
- 그래픽카드(GPU) 칩셋·메모리 정보 조회  
- CPU 벤치마크 기능 제공  
- 설치형 + 무설치(Portable) 버전 지원  
- Android 앱 제공 (모바일 사양 확인 가능)
